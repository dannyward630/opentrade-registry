# Blocked Source Reassessment

v1 has no provisional adapter queue. Every source is implemented or terminally blocked/deprecated. The generated [source-status matrix](../source-status-matrix.md) is authoritative.

A blocked source may be reconsidered when official evidence changes. Promotion requires:

- a lawful, stable file, API, or lookup contract;
- no CAPTCHA, account, paywall, or technical-control bypass;
- reviewed terms and redistribution posture;
- contractor/trade-specific scope and exclusions;
- a tiny hand-authored fixture using official field names;
- parser, mapper, status, duplicate, malformed-row, and verification tests;
- Level 4 caveat review;
- local-file or explicit-network capability rather than a fixture-only terminal state.

Do not choose work from hints in prose. Update source evidence first, change the terminal outcome only when implementation passes all gates, and regenerate the source matrix and hosted seed.
