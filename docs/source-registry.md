# Source Registry

The source registry is a versioned catalog of official public sources. It does not imply that generated datasets can be redistributed.

Every entry includes:

- `id`
- `name`
- `jurisdiction`
- `agency`
- `sourceType`
- `sourceUrl`
- `documentationUrl`
- `dataDictionaryUrl`
- `termsUrl`
- `updateFrequency`
- `tradeCoverage`
- `licenseTypesIncluded`
- `knownExclusions`
- `hasBulkDownload`
- `hasLiveLookup`
- `requiresJavaScript`
- `requiresCaptcha`
- `requiresAccount`
- `rateLimitNotes`
- `redistributionStatus`
- `publicRecordsNotes`
- `adapterStatus`
- `adapterPackage`
- `testFixturePath`
- `lastVerifiedAt`
- `maintainerNotes`

Use `redistributionStatus: "unknown"` unless redistribution rights are clearly confirmed.
