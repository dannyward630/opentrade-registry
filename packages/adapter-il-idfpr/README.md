# Archived Illinois Parser Research

This private workspace package preserves historical Illinois IDFPR parser research. It is not published, not registered by the CLI, and not a supported adapter.

The tiny hand-authored fixture does not establish compatibility with an official file or lookup response. No stable public IDFPR bulk file shape was validated during the v1 review.

The public source registry therefore records `us.il.idfpr.roofing_contractors` as `blocked`. Do not use this package for licensing conclusions.
